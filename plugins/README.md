# Development Plug-in Modules

This directory contains optional pluggable modules and features that are not essential for the core RAG functionality but might be integrated in the future.

## Contents

- Alternative embedding providers (Ollama)
- Additional rerankers (Ollama-based reranking)
- Web search integration
- Experimental retrieval strategies

These modules are maintained separately from the main codebase to keep the core pipeline clean and focused on essential functionality.

## Usage

If you want to integrate any of these features, please refer to the module-specific documentation and ensure all dependencies are installed before integration.
